# Kuldeep Singh — Cybersecurity Portfolio Website

## 🚀 How to Use

### Add Your Photo
1. Place your photo file inside the `assets/` folder
2. Rename it to `photo.jpg`
3. That's it — the website will show it automatically!

If no photo is found, the site shows your initials "KS" as a stylish fallback.

### Open the Website
Just double-click `index.html` to open it in your browser.
No server needed!

### 🌐 Host on GitHub Pages (Free!)
1. Create a GitHub account at github.com
2. Create a new repository named: `kuldeep-portfolio` or `cyber-orbit.github.io`
3. Upload all these files to the repo
4. Go to: Settings → Pages → Branch: main → Save
5. Your site goes live at: `https://cyber-orbit.github.io`

## 📁 File Structure
```
kuldeep-portfolio/
├── index.html      ← Main website file
├── style.css       ← All styles and design
├── app.js          ← Animations and interactions
├── assets/
│   └── photo.jpg   ← YOUR PHOTO GOES HERE
└── README.md       ← This file
```

## ✏️ Customizing
- **Edit text**: Open `index.html` in any text editor (Notepad, VS Code)
- **Change colors**: Open `style.css` and edit the `:root` variables at the top
- **Add projects**: Copy a `.proj-card` block in `index.html` and edit it
